# Privacy Policy

This project was created as a personal digital gift.

This app itself does not collect, sell, or share personal data.

If a password is entered, it is only used to unlock the experience. The app does not store the password submission.

If dynamic message generation is enabled, the app may send a generic prompt to the configured AI API provider to generate a message. The password is not sent to the AI API provider by this app.

No analytics, tracking, or advertising tools are included in this project.
